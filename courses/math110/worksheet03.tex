{\bf MATH 110 Linear Algebra}

\begin{enumerate}
\item {\em True or False?} $\mathbb{C}$ is a vector space over $\mathbb{Q}$

\item Consider the following two subsets of $\RR^\RR$:
\[
	\mathcal{O} = \{f: f \text{ is odd}\} \qquad \text{ and } \qquad \mathcal{E} = \{f: f \text{ is even}\}
\]
\begin{enumerate}
\item Show that $\mathcal{O}$ and $\mathcal{E}$ are vector spaces over $\RR$
\item What is $\mathcal{O} + \mathcal{E}$?
\item Is that a direct sum?
\end{enumerate}

\item {\em True or False?} Consider $\mathbb{R}^2$ as a vector space over $\mathbb{R}$. $(\sqrt{2}, \sqrt{6})$ and $(-1, -\sqrt{3})$ are linearly independent

\item {\em True or False?} Same as above, but we consider them as vectors over $\mathbb{Q}$ instead.

\item Suppose $\{v_1, v_2, v_3\}$ forms a basis for a vector space $V$ over $\mathbb{F}$. Does $\{v_1 + v_2, v_2 + v_3, 2v_1 + v_2 + 2v_3\}$ form a basis?

\item {\em (Challenging)} Consider the following vector space over $\RR$
\[
	S = \left\{(a_i)_{i = 1, 2, 3, \ldots}: a_i \text{ is a real sequence with } \sum_{i=1}^\infty a_i \text{ converges}\right\}.
\]
Show that it is infinite dimensional. Does
\[
	\{(1, 0, 0, \ldots), (1, 1, 0, 0, \ldots), (1, 1, 1, 0, 0, \ldots), \ldots\}
\]
form a basis?

\item If the statement in Question 1 is true, what is the dimension?

\item {\em (Challenging)} Consider the vector space over $\RR$ (and convince yourself this is a vector space!)
\[
	P = \{f:f \text{ is a periodic function from } \ZZ \text{ to } \RR\}
\]
What is the dimension of $P$?
\end{enumerate}
